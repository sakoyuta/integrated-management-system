# 営業育成管理ダッシュボード

React ベースの営業担当者向け教育・進捗管理システム

## 🚀 デモサイト

https://sakoyuta.github.io/management1/

## 📋 機能概要

### 基本機能
- ✅ **14人の営業担当者管理** - タブ形式で切り替え
- ✅ **21項目のトレーニング管理** - カテゴリ別に分類
- ✅ **4段階の進捗ステータス** - 未着手 / ✖ / △ / ◎
- ✅ **円形プログレスバー** - 視覚的な達成度表示
- ✅ **コメント履歴機能** - 日付付きフィードバック管理

### 管理機能（管理者モード）
- ✅ **管理者認証** - パスワード: `house06`
- ✅ **営業担当者の追加・削除**
- ✅ **トレーニング項目の追加・削除**
- ✅ **PDF資料の添付・管理** - ドラッグ&ドロップ対応
- ✅ **編集ボタン** - 各項目カードに編集機能
- ✅ **管理者パネル** - 折りたたみ式の管理インターフェース

### UI/UX機能
- ✅ **レスポンシブデザイン** - モバイル・デスクトップ対応
- ✅ **Toast通知システム** - 操作フィードバック
- ✅ **コメント履歴の展開/折りたたみ**
- ✅ **ホバーエフェクト** - インタラクティブなUI

## 📦 セットアップ

### 1. 簡単な起動方法
```bash
# HTMLファイルを直接ブラウザで開く
open completed-sales-dashboard.html
```

### 2. ローカルサーバーでの起動
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx serve .

# ブラウザで http://localhost:8000 にアクセス
```

## 🔑 使用方法

### 基本操作
1. ブラウザで `completed-sales-dashboard.html` を開く
2. 営業担当者タブで担当者を切り替え
3. 各トレーニング項目の進捗を確認

### 管理者モード
1. 「管理者モード」ボタンをクリック
2. パスワード `house06` を入力
3. 管理者パネルで以下の操作が可能：
   - 担当者の追加・削除
   - トレーニング項目の追加・削除
   - PDF資料の添付・編集
   - 進捗ステータスの更新
   - コメント履歴の管理

### PDF資料管理
1. 管理者モードで各項目の編集ボタン（鉛筆アイコン）をクリック
2. PDFファイルをドラッグ&ドロップまたはクリックで選択
3. ファイルサイズ制限: 10MB以下
4. 対応形式: PDFのみ

## 🎯 進捗管理システム

### ステータス定義
- **未着手** (0%) - まだ開始していない
- **✖** (10%) - ロープレが出来ず知識が不足
- **△** (50%) - ロープレができる状態だが不合格
- **◎** (100%) - 部門長のロープレ合格済

### カテゴリ分類
- **商談スキル** - ヒアリング、上位概念、競合対策など
- **資金計画** - 資金計画書、トータルコスト、補助金など
- **商品知識** - 構造、断熱、気密、太陽光など
- **土地知識** - 土地関連のトーク
- **設計** - 間取り、デザイン
- **顧客関係** - 保証関連

## 🔒 セキュリティ

- クライアントサイドのみの実装（サーバー不要）

## 🐛 トラブルシューティング

### PDFが表示されない
- ブラウザのポップアップブロックを無効化
- ファイルサイズが10MB以下か確認
- PDF形式か確認

### 管理者モードにアクセスできない
- パスワード `house06` を正確に入力
- ブラウザの JavaScript が有効か確認

### レスポンシブが正しく動作しない
- ブラウザの開発者ツールでビューポートを確認
- Tailwind CSS が正しく読み込まれているか確認
